**🚀 AI-Powered Sales Business Intelligence System**

An end-to-end Business Intelligence and AI analytics system that combines **Power BI, Python, Machine Learning, and Streamlit** to deliver automated insights, forecasting, and executive-level reporting.

**📊 Project Overview**

This project analyzes retail sales data (5,000+ records) and transforms it into an interactive, AI-driven business intelligence platform.

The system includes:

🔹 Advanced Power BI dashboards (Executive Overview, Sales Analytics, Risk & Performance, AI Insights)

🔹 Star schema data modeling with DAX measures

🔹 KPI engine for automated metric generation

🔹 Anomaly detection using statistical Z-score method

🔹 Machine Learning-based monthly sales forecasting (Linear Regression)

🔹 AI-powered question-answer assistant (Streamlit)

🔹 Automated executive PDF report generation (ReportLab)

**🏗 Architecture**

Power BI → Data Modeling & Visualization

Python (Pandas, Scikit-learn) → Analytics Engine

Streamlit → AI Web Interface

ReportLab → Automated PDF Reports

**🔍 Key Features**

✔ 10+ dynamic KPIs

✔ Year-over-Year growth analysis

✔ Forecast with 95% confidence interval

✔ Risk detection (loss-making products & high discounts)

✔ Drill-through & tooltip-enabled dashboards

✔ Natural language business queries

✔ One-click executive report generation

**📈 Business Impact**

Reduced manual analysis effort

Automated reporting workflow

Enabled faster executive decision-making

Converted static dashboard into AI-driven BI assistant
